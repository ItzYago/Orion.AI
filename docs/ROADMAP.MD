# Orion Roadmap

O roadmap da Orion define as funcionalidades planejadas e os objetivos de desenvolvimento para as próximas versões do projeto.

---

## Fase 1 — Fundação

### v0.1.0

* [x] Estrutura inicial do projeto
* [x] Definição da identidade da Orion
* [x] Criação da documentação inicial
* [x] Configuração do ambiente de desenvolvimento

### v0.1.1

* [x] Interface básica via terminal
* [x] Sistema de entrada e saída
* [x] Comando de ajuda (`help`)
* [x] Comando de encerramento (`exit`)

### v0.1.2

* [x] Sistema de comandos
* [x] Parser de argumentos
* [x] Tratamento de erros
* [x] Organização dos módulos internos
