# Changelog

Todas as mudanças notáveis do projeto Orion serão documentadas neste arquivo.

O formato segue um controle simples de versionamento para acompanhar a evolução do projeto.

---

## [v0.1.0] - Estrutura Inicial

### Adicionado

* Estrutura inicial do projeto.
* Arquivo README.md.
* Arquivo requirements.txt.
* Sistema inicial de configuração.
* Definição da identidade da Orion.
* Documentação legal e avisos de uso.
* Planejamento da arquitetura do projeto.

### Objetivo da versão

Estabelecer a fundação da Orion e preparar o ambiente para o desenvolvimento das funcionalidades principais.
